# quickstart-demo — Idea Board

## Format
Every idea has: ID (`<PREFIX>-NNN`, sequential, never reused), Status emoji
(📋 proposed · 🔨 in-progress · ✅ done · 🛑 deferred · 🗑️ rejected),
Source (wikilinks), Title, Effort (XS/S/M/L/XL), Impact (H/M/L).

Detail blocks live in `IDEAS-details/<batch-name>.md` — written on demand,
not eagerly. See `wiki/SCHEMA.md` for the detail-block template.

## Numbering policy
- Sequential within a batch; gaps allowed between batches.
- Never reuse a number, even for rejected ideas.
- Document gaps explicitly.
